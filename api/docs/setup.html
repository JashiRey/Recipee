Recipe API Setup Guide

1. Install XAMPP

  1. Visit the XAMPP download page
  2. Download the version appropriate for your operating system:
    • Windows: XAMPP for Windows
    • macOS: XAMPP for OS X
    • Linux: XAMPP for Linux
  3. Run the installer and follow the installation wizard
  4. For macOS users:
    • Open the downloaded .dmg file
    • Double-click the XAMPP installer package
    • Follow the installation steps
    • XAMPP will be installed in /Applications/XAMPP

2. Get the Repository

  1. Choose one of these methods:

    Option A: Using Git

    1. Open Terminal or Command Prompt
    2. Navigate to any directory where you want to clone the project 
      cd /path/to/your/preferred/location
    3. Clone the repository: 
      git clone https://github.com/JashiRey/Recipee.git

    Option B: Direct Download

    1. Visit https://github.com/JashiRey/Recipee
    2. Click the green "Code" button and select "Download ZIP"
    3. Extract the ZIP file to any location on your computer
  2. Copy or move the project folder to XAMPP's htdocs:

    For macOS:

    mv /path/to/downloaded/Recipee /Applications/XAMPP/xamppfiles/htdocs/recipee

    Or use Finder to copy the folder to: /Applications/XAMPP/xamppfiles/htdocs/recipee

    For Windows:

    move C:\path\to\downloaded\Recipee C:\xampp\htdocs\recipee

    Or use File Explorer to copy the folder to: C:\xampp\htdocs\recipee

  3. Verify that the project folder is named exactly recipee in the htdocs directory
Important: Make sure the final path is:
  • macOS: /Applications/XAMPP/xamppfiles/htdocs/recipee
  • Windows: C:\xampp\htdocs\recipee
The folder name must be exactly "recipee" (lowercase) for the API to work correctly.

3. Configure Apache

  1. Open XAMPP Control Panel
  2. Click on "Config" next to Apache
  3. Select "httpd.conf"
  4. Find the following lines and modify them: 
    DocumentRoot "/Applications/XAMPP/xamppfiles/htdocs/recipee"
<Directory "/Applications/XAMPP/xamppfiles/htdocs/recipee">
  5. Save the file
  6. Restart Apache using XAMPP Control Panel
Important: Make sure the path matches your actual installation path. Windows users will need to use Windows-style paths: 
DocumentRoot "C:/xampp/htdocs/recipee"
<Directory "C:/xampp/htdocs/recipee">

4. Set Up the Database

  1. Start both Apache and MySQL in XAMPP Control Panel
  2. Open your browser and navigate to http://localhost/phpmyadmin
  3. Click on "New" to create a new database
  4. Name it "recipee" and click "Create"
  5. Click on the "Import" tab
  6. Download the database script: setup-script.sql
  7. Click "Choose File" in phpMyAdmin and select the downloaded script
  8. Click "Go" to import the database structure

5. Verify Installation

  1. Open your browser
  2. Navigate to http://localhost/api/docs/
  3. You should see the API documentation page
  4. Try the API playground at http://localhost/api/playground/

Troubleshooting

  • Make sure both Apache and MySQL are running in XAMPP Control Panel
  • Check Apache error logs if you encounter issues
  • Verify file permissions are correct
  • Make sure all paths in httpd.conf match your actual installation
  • If you get database connection errors, verify your database name and credentials
