Recipee API Documentation

Users

GET /api/users/

Get all users or filter by parameters

Query Parameters:

  • id - Filter by user ID
  • email - Filter by exact email match

Example URLs:

  • /api/users/ - Get all users
  • /api/users/?id=1 - Get user with ID 1
  • /api/users/?email=john@example.com - Get user by email

Response:

[
    {
        "id": 1,
        "name": "John Doe",
        "email": "john@example.com"
    }
]
POST /api/users/

Create a new user

Request Body:

{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "secretpassword"
}

Response:

{
    "message": "User created successfully",
    "user_id": 1
}
POST /api/users/auth/

Authenticate user

Request Body:

{
    "email": "john@example.com",
    "password": "secretpassword"
}

Response:

{
    "message": "Authentication successful",
    "user": {
        "id": 1,
        "name": "John Doe",
        "email": "john@example.com"
    }
}
PUT /api/users/

Update user information

Request Body:

{
    "id": 1,
    "name": "John Updated",    // optional
    "email": "new@email.com",  // optional
    "password": "newpassword"  // optional
}

Response:

{
    "message": "User updated successfully",
    "user_id": 1
}
DELETE /api/users/?id=1

Delete a user

Example URL:

  • /api/users/?id=1 - Delete user with ID 1

Response:

{
    "message": "User deleted successfully",
    "user_id": 1
}

Recipes

GET /api/recipes/

Get all recipes or filter by parameters

Query Parameters:

  • id - Filter by recipe ID
  • user_id - Filter by user ID
  • name - Filter by recipe name (partial match)
  • ingredient_ids - Filter by ingredient IDs (comma-separated)

Example URLs:

  • /api/recipes/ - Get all recipes
  • /api/recipes/?id=1 - Get recipe with ID 1
  • /api/recipes/?user_id=1 - Get all recipes by user 1
  • /api/recipes/?name=pasta - Search recipes containing "pasta"
  • /api/recipes/?ingredient_ids=1,2,3 - Get recipes containing ingredients 1, 2, and 3

Response:

[
    {
        "id": 1,
        "user_id": 1,
        "name": "Spaghetti Carbonara",
        "content": "Recipe instructions...",
        "imgurl": "https://example.com/image.jpg",
        "ingredients": [
            {
                "ingredient_id": 1,
                "name": "Spaghetti",
                "quantity": 500,
                "unit": "g",
                "format": "decimal"
            }
        ]
    }
]
POST /api/recipes/

Create a new recipe

Request Body:

{
    "user_id": 1,
    "name": "Spaghetti Carbonara",
    "content": "Recipe instructions...",
    "imgurl": "https://example.com/image.jpg",
    "ingredients": [
        {
            "ingredient_id": 1,
            "quantity": 500,
            "unit": "g",
            "format": "decimal"
        }
    ]
}

Response:

{
    "message": "Recipe created successfully",
    "recipe_id": 1
}
DELETE /api/recipes/?id=1

Delete a recipe

Example URL:

  • /api/recipes/?id=1 - Delete recipe with ID 1

Response:

{
    "message": "Recipe deleted successfully",
    "recipe_id": 1
}

Ingredients

GET /api/ingredients/

Get all ingredients or filter by parameters

Query Parameters:

  • id - Filter by ingredient ID
  • name - Filter by ingredient name (partial match)

Example URLs:

  • /api/ingredients/ - Get all ingredients
  • /api/ingredients/?id=1 - Get ingredient with ID 1
  • /api/ingredients/?name=oil - Search ingredients containing "oil"

Response:

[
    {
        "id": 1,
        "name": "Olive Oil"
    }
]
POST /api/ingredients/

Create a new ingredient

Request Body:

{
    "name": "Olive Oil"
}

Response:

{
    "message": "Ingredient created successfully",
    "ingredient_id": 1
}
PUT /api/ingredients/

Update ingredient information

Request Body:

{
    "id": 1,
    "name": "Updated Olive Oil"
}

Response:

{
    "message": "Ingredient updated successfully",
    "ingredient_id": 1
}
DELETE /api/ingredients/?id=1

Delete an ingredient (only if not used in any recipes)

Example URL:

  • /api/ingredients/?id=1 - Delete ingredient with ID 1

Response:

{
    "message": "Ingredient deleted successfully",
    "ingredient_id": 1
}
